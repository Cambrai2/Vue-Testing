Github Top Public Repositories

Please find below a list of the top 100 public Github repositories. The data was compiled via an Axios get function to the Github API page (https://api.github.com/repositories). A Vue instance is then instantiated and the Table populated before being turned into a functioning and interactable datatable using Bootstrap4. The datatable allows for searching, sorting, changing number of entries and pagination.
